Lessons Learned While Learning to Code Using HTML and CSS

Introduction

This is a compilation of lessons I've learned while learning to code in HTML and CSS. In the future I may expand it to include Javascript.

Although I explored, and tinkered with front-end web development in the past, I never stuck with it. Then in about 2018 I discovered the freeCodeCamp site and curriculum. For some reason it made all the difference in my interest level and desire to continue learning.

I'm creating this page to fulfill one of the freeCodeCamp front-end development assignments, and as a journal of techniques for my own reference.
