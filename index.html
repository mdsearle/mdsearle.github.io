Preface

For several years I've been trying to further learn HTML and CSS; to develop and improve my ability to create webpages using both languages. During that time I've used many sources for my study. However, the main two sources are Free Code Camp and the Mozilla Developer Network. Although both are dedicated to teaching HTML and CSS, I've learned unique principles from each one. And, I would recommend both sources to anyone else wanting to learn to create webpages using HTML and CSS.

This page serves as an index to most of the webpages I've created during my studies using the tutorials from both sources.
